## Accessing the Help Menu

Press `F1` to show the server help and information pages. 

## What's Inside?

From the menu you will be able to access:

- Help
  - settings
  - rates
  - rules
- Information
  - vote and claim extra shop points
  - view stats 
  - view achievements
  - setup offline raid protection (ORP)
  - auto-fill turrets
